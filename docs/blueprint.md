# **App Name**: Ignitia

## Core Features:

- User Authentication: Secure sign-up, login, and role-based access control (learner, mentor, admin).
- User Profiles: User profiles tracking personal details, skills, learning goals, and progress.
- Skill Assessment: Skill diagnostic tests with AI-powered evaluation to generate skill profiles using reasoning tool.
- Adaptive Learning Paths: Dynamically recommended personalized learning paths with curated content, mini projects, and exam modules.
- Course Integration: Integrate APIs from major education platforms (Coursera, Udemy, etc.) for seamless course enrollment.
- Progress Visualization: Interactive dashboards showing skill mastery, learning milestones, and time spent.
- Practical Application: Mini projects and quizzes for applying new skills, with feedback systems to guide improvement.

## Style Guidelines:

- Primary color: Soft beige (#E6DFD1) as the primary background for a calm and inclusive feel.
- Background color: Muted gray (#86867c) for UI elements, maintaining consistency.
- Accent color: Deep brown (#4A3C30) for text, dividers, and interactive UI states, providing a sophisticated contrast.
- Font pairing: 'Alegreya', a serif font for headers and 'PT Sans', a sans-serif font for body text, ensuring readability and an elegant feel.
- Dashboard: Sectioned cards and grid-based layout with welcome area, skill assessment overview, learning path, progress visualization, and notifications.
- Use integrated icons aligned with the color tone for easy navigation and understanding.
- Subtle hover effects on buttons and gentle transitions for a smooth user experience.
